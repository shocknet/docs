# About Lightning.Pub

Share your node with friends, family, apps, and customers over Nostr with the first Nostr-based account system for Lightning


### install

Default installation cli lauch script

```
wget -qO- https://deploy.lightning.pub | bash
```

script for testing version with an added log along installation
```
wget -qO- https://tdeploy.lightning.pub | bash
```

run this to output a log detailing what happened
```
cat /tmp/lightning_pub_install.log
```
