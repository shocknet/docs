# Deploy on Start9

Lightning.Pub can be sideloaded onto Start9 OS, it is not yet available in the marketplace as development and testing is highly active.

## Installation

Lightning.Pub is a wrapper over LND, ensure you have LND up and running on your Start9.

1. Download the latest `.s9pk` package from [GitHub Releases](https://github.com/shocknet/Lightning.Pub/releases)
2. In your Start9 interface, go to **System** > **Sideload a Service**
3. Upload the `.s9pk` file and click "Sideload"
4. Start the service

## Connect

After starting:
1. Open the Lightning.Pub service in Start9
2. Launch UI
3. Use the wizard to configure your preferences then scan the nprofile QR code or paste into ShockWallet to connect a device.
